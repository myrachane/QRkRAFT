# QR KRAFT

A customizable QR code generator built with Electron. Generate QR codes with custom module shapes, finder pattern styles, colors, transparent backgrounds, and logo overlays.

## Features

- **Module Shapes** — Square, Circle, X, Plus
- **Finder Styles** — Standard, Rounded, Circle, Diamond, Leaf, Mixed, Frame, Dots
- **Colors** — Custom foreground and background colors
- **Transparent Background** — Export with alpha transparency
- **Logo Overlay** — Embed any image in the center
- **Error Correction** — L / M / Q / H levels
- **Export** — PNG image or SVG vector

## Download

Grab the latest installer from the [releases page](https://github.com/myrachane/qrkraft/releases) or build from source.

## Build from Source

```sh
git clone https://github.com/myrachane/qrkraft.git
cd qrkraft
npm install
npm start          # run in development
npm run dist       # build installer
```

## Usage

1. Type text or a URL into the content field
2. Customize the appearance using the sidebar controls
3. Preview updates in real-time
4. Export as PNG or SVG
